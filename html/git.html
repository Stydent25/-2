Git

  • git clone расшифровывается как "clone" ,что переводится как создание локальной копии удаленного или существующего репозитория
  • git add расшифровывается как (add),что переводится как добавления изменений из рабочего каталога
  • git status расшифровывается как "status",что переводится как отображения текущего состояния вашего Git-репозитория, показывая изменения в рабочем каталоге и области индексации
  • git commit расшифровывается как "commit",что переводится как изменения в вашем проекте, создавая "снимок" состояния файлов в определенный момент времени
  • git push расшифровывается как "push",что означает отправку (переноса) локальных изменений из вашего репозитория на удалённый сервер.
  • git pull Расшифровывается как "pull", что означает как извлечение и загрузку изменений из удаленного репозитория и их последующее слияние с вашей локальной копией проекта, синхронизируя ее с удаленной версией.
